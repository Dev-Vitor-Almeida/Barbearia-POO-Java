# Sistema de Agendamento para Barbearia 💈

Projeto acadêmico desenvolvido com foco em Programação Orientada a Objetos (POO), simulando o funcionamento de um sistema de gerenciamento e agendamento para uma barbearia.

## 🚀 Funcionalidades atuais

* Cadastro de clientes
* Cadastro de barbeiros
* Cadastro de serviços
* Criação de agendamentos
* Listagem de dados utilizando ArrayList
* Estrutura inicial de CRUD
* Relacionamento entre classes
* Exibição de informações específicas para clientes e barbeiros

## 🧠 Conceitos aplicados

* Encapsulamento
* Herança
* Abstração
* Polimorfismo
* Organização em pacotes
* Programação Orientada a Objetos (POO)

## 🛠 Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Git & GitHub

## 📌 Próximas implementações

* Integração com banco de dados MySQL
* CRUD completo
* Interface gráfica / Front-end
* Sistema de login
* Dashboard administrativo
* Validação de horários disponíveis
* Melhorias na arquitetura do sistema

## 📚 Objetivo do projeto

O projeto tem como objetivo aplicar conceitos de desenvolvimento back-end e Programação Orientada a Objetos em uma solução voltada para automação de pequenos negócios.
